# GamingRent 

## ¿Qué es?
**GamingRent Backoffice** es una aplicación web interna para gestionar una **tienda de alquiler de videojuegos y consolas**.  
Está desarrollada en **PHP** y conectada a una base de datos **MySQL** .

## ¿Para qué sirve?
Permite al personal del negocio gestionar la información principal del día a día y consultar datos útiles para la toma de decisiones.

## Funciones principales
- **Dashboard**
  - Resumen rápido del estado del negocio (clientes, productos, alquileres, etc.).
  - Acceso a rankings y datos destacados (según vistas/consultas disponibles).

- **Gestión de clientes**
  - Listado de clientes.
  - Búsqueda por nombre.
  - Alta y edición de datos.
  - Eliminación (según implementación/configuración del proyecto).

- **Gestión de productos y stock**
  - Listado de productos (videojuegos y consolas).
  - Control de stock/disponibilidad.
  - Alta, edición y eliminación (según implementación/configuración del proyecto).

- **Alquileres**
  - Registro y consulta de alquileres realizados.
  - Control de fechas (inicio/fin) y estado del alquiler.

- **Pagos**
  - Registro y consulta de pagos asociados a alquileres.
  - Seguimiento básico de importes.

- **Empleados / Usuarios**
  - Acceso por roles (según configuración).
  - Pantallas de administración para gestión interna.

- **Reportes (vistas)**
  - Pantallas de consulta basadas en **vistas SQL** para informes rápidos.
  - Ejemplo: productos más alquilados, clientes con más actividad, etc. (según vistas disponibles).

- Integración PHP–MySQL con PDO
- Estructura típica de un backoffice
